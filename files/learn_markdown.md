# Markdown記法を学びましょう

Markdownは20年以上前に公開された規格で、多分AIエージェントやCLI型のAIが浸透し始めた時期に急速に普及しました。
HTMLより手軽に書けて、最低限で十分な表現力を持ち合わせています。

## 1. 見出し

---

見出しには`#`を使います。
`#`から`######`まで対応しており、`#`が少ないほど大きな見出しとなります。

```markdown
# 最大
## 大
### 中
#### 小
##### さらに小
###### 最小
```

これは以下の様に変換されます。

# 最大
## 大
### 中
#### 小
##### さらに小
###### 最小

`#`のあとに空白を入れないと適用されないことに注意してください。

## 2. テキスト

---

### 段落

段落を付けたいとき、HTMLのようになにか付ける必要はなく、ただ改行を1つ挟めばよいです。

```HTML
<p>テキストテキスト1</p>
<p>テキストテキスト2</p>
```

は

```markdown
テキストテキスト1

テキストテキスト2
```

のように再現します。

### 装飾

HTML同様に装飾ができます。

|装飾|Markdown|HTML|
|---|---|---|
|ボールド|`**テキスト**`|`<strong>テキスト</strong>`|
|イタリック|`_テキスト_`|`<em>テキスト</em>`|
|コード|\`テキスト\`|`<code>テキスト</code>`|

```markdown
**ボールド**

_イタリック_

`コード`
```

**ボールド**

_イタリック_

`コード`

## 3. リストとチェックボックス

### アンオーダーリスト

ただのリストは`- `を先頭に付けて表現します。

```markdown
- 項目
- 項目
```

- 項目
- 項目

### オーダーリスト

数字のついたリストは`1. `や`2. `を先頭に付けて表現します。

```markdown
1. 項目1
2. 項目2
```

1. 項目1
2. 項目2

### ネストさせる

これらはどんどん深く入れ子にすることができます。

```markdown
- 買うもの
    1. リンゴ
        - 赤いリンゴ
        - 3つ
            - 予算を超えたら減らしてOK
    2. ジュース
        1. コーラ
            - ペプシ
        2. オレンジジュース
            - 100%
```

- 買うもの
    1. リンゴ
        - 赤いリンゴ
        - 3つ
            - 予算を超えたら減らしてOK
    2. ジュース
        1. コーラ
            - ペプシ
        2. オレンジジュース
            - 100%

### チェックボックス

`- [ ]`でチェックボックスを作ることができます。`[ ]`にxをいれて`[x]`にすることで、チェックを付けることができます。

```markdown
- [ ] 項目1
- [x] 項目2
- [ ] 項目3
```

- [ ] 項目1
- [x] 項目2
- [ ] 項目3
