# Markdown記法を学びましょう

Markdownは20年以上前に公開された規格で、多分AIエージェントやCLI型のAIが浸透し始めた時期に急速に普及しました。
HTMLより手軽に書けて、最低限で十分な表現力を持ち合わせています。
ほとんどのAIチャットアプリがこの規格を使ってスタイリングしています。

## 1. 見出し

---

見出しには`#`を使います。
`#`から`######`まで対応しており、`#`が少ないほど大きな見出しとなります。

```markdown
# 最大
## 大
### 中
#### 小
##### さらに小
###### 最小
```

↓

# 最大
## 大
### 中
#### 小
##### さらに小
###### 最小

`#`のあとに空白を入れないと適用されないことに注意してください。

## 2. テキスト

---

### 段落

段落を付けたいとき、HTMLのようになにか付ける必要はなく、ただ改行を1つ挟めばよいです。

```HTML
<p>テキストテキスト1</p>
<p>テキストテキスト2</p>
```

は

```markdown
テキストテキスト1

テキストテキスト2
```

のように再現します。

### 装飾

HTML同様に装飾ができます。

|装飾|Markdown|HTML|
|---|---|---|
|ボールド|`**テキスト**`|`<strong>テキスト</strong>`|
|イタリック|`_テキスト_`|`<em>テキスト</em>`|
|コード|\`テキスト\`|`<code>テキスト</code>`|
|打ち消し線|`~~テキスト~~`|`<s>テキスト</s>`|

```markdown
**ボールド**

_イタリック_

`コード`

~~打ち消し線~~
```

↓

**ボールド**

_イタリック_

`コード`

~~打ち消し線~~

## 3. リストとチェックボックス

---

### アンオーダーリスト

ただのリストは`- `を先頭に付けて表現します。

```markdown
- 項目
- 項目
```

- 項目
- 項目

### オーダーリスト

数字のついたリストは`1. `や`2. `を先頭に付けて表現します。

```markdown
1. 項目1
2. 項目2
```

1. 項目1
2. 項目2

### ネストさせる

これらはどんどん深く入れ子にすることができます。

```markdown
- 買うもの
    1. リンゴ
        - 赤いリンゴ
        - 3つ
            - 予算を超えたら減らしてOK
    2. ジュース
        1. コーラ
            - ペプシ
        2. オレンジジュース
            - 100%
```

- 買うもの
    1. リンゴ
        - 赤いリンゴ
        - 3つ
            - 予算を超えたら減らしてOK
    2. ジュース
        1. コーラ
            - ペプシ
        2. オレンジジュース
            - 100%

### チェックボックス

`- [ ]`でチェックボックスを作ることができます。`[ ]`にxをいれて`[x]`にすることで、チェックを付けることができます。

```markdown
- [ ] 項目1
- [x] 項目2
- [ ] 項目3
```

- [ ] 項目1
- [x] 項目2
- [ ] 項目3

## 4. テーブル

---

テーブルもサクッと作ることができます。
少しややこしいので見て覚えてください。

```markdown
|氏名|国語|算数|
|---|---|---|
|田中太郎|80|20|
|山田太郎|30|70|
|鈴木太郎|50|50|
```

↓

|氏名|国語|算数|
|---|---:|---:|
|田中太郎|80|20|
|山田太郎|30|70|
|鈴木太郎|50|50|

一行目で列を定義し、二行目でどちらに寄せるかを決め(---は左寄せ、---:は右寄せ)、三行目以降で具体的なデータを入れていきます。

## 5. 数式

---

LaTeX記法に対応しています。
`$$数式$$`でブロック、`$数式$`でインラインとなります。

```markdown
$$p = \rho gh$$

- $p$: パスカル
- $\rho$: 流体の密度
- $g$: 重力加速度($9.8 \text{ m/s}^2$)
- $h$: 基準面からの深さ、または流体の高さ
```

↓

$$p = \rho gh$$

- $p$: パスカル
- $\rho$: 流体の密度
- $g$: 重力加速度($9.8 \text{ m/s}^2$)
- $h$: 基準面からの深さ、または流体の高さ

## 6. その他

---

- `---`で水平線
- `> テキスト`で引用風
- `![代替テキスト](画像URL)`で画像表示

など、他にも様々な機能があります。

end
